🔧 Configurazione Firebase Richiesta

Per utilizzare l'applicazione con sincronizzazione online, segui questi passi:

  1. Vai su Firebase Console
  2. Crea un nuovo progetto (es: "desiderata-reparto")
  3. Aggiungi un'app Web al progetto
  4. Copia la configurazione Firebase
  5. Apri questo file HTML con un editor di testo
  6. Cerca la sezione FIREBASE_CONFIG (circa riga 600)
  7. Incolla i tuoi valori di configurazione
  8. Abilita "Firestore Database" nel progetto Firebase (modalità test)
  9. Ricarica questa pagina

📧 Per assistenza nella configurazione, contatta il supporto IT.
